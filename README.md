# Skyblock
![alt text](https://apex-misc.s3.amazonaws.com/Images/Skyblock+Thumbnail.png)

Skyblock is one of the classic gamemods preferred by the minecraft community.
 
You start on tiny islands in the sky, all necessary items, blocks and materials are located on the island's chest, with this items you will need to survive and develop your island!
